Tentang MySQL

Teori MySQL

MySQL adalah sistem manajeman basis data relasional (Relational Database Management System/RDBMS) yang menunggunakan SQL (Structured Query Languange) untuk mengelola data.

Karakteristik MySQL

  • Open Source kode yang bersifat terbuka, mudah diakses, digunakan serta disebarluaskan oleh siapapun
  • Cross-platform Bisa berjalan di berbagai OS.
  • High Performance mampu memproses data dengan cepat dan efisien.
  • Scalability Mampu menangani database kecil hingga skala enterprise.
  • Security Memiliki sistem user dan hak akses yang bisa diatur.

Fungsi Utama MySQL

  • Menyimpan data dalam bentuk tabel
  • Mengelola data (menambah, mengubah, menghapus)
  • Menyediakan akses data yang cepat dan aman
  • Menjalankan Query untuk menampilkan data sesuai kebutuhan
  • Digunakan sebagai Backend database untuk aplikasi web maupun dekstop

Komponen Utama MySQL

  • Database Kumpulan tabel.
  • Tabel Menyimpan data dalam baris (row) dan kolom (column)
  • Field / Kolom Atribut dari data (contoh: nama, harga).
  • Record / Baris Data Aktual dalam tabel.
  • Primary Key Identitas unik untuk tiap record
  • foreign Key Kunci untuk menghubungkan tabel.
