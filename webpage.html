APA YANG KAMU HARAPKAN DARI MK BASIS DATA?

Di era digital saat ini, data menjadi salah satu hal yang sangat berharga. hampir semua aplikasi, mulai dari media sosial, sistem pembayaran hingga aplikasi belanja online memerlukan database sebagai tempat pengelolaan dan penyimpanan data. karena itu, saya merasa sangat penting untuk mempelajari database agar dapat memahami bagaimana data disimpan, dikelola serta dimanfaatkan dengan baik.

Sebagai mahasiswa informatika, saya ingin menyadari bahwa belajar tentang database bukan hanya untuk menyelesaikan tugas, tetapi juga menjadi keterampilan penting yang relavan dalam berkarir di dunia kerja baik di industri maupun digital. oleh karena itu, saya memiliki beberapa harapan besar dalam mempelajari database.

pertama, saya berharap saya mampu mempelajari dan memahami dengan baik tentang dasar database. Mulai dari baris, kolom, atribute, hingga hubungan antar tabel.

kedua, saya ingin menguasai bahasa SQL untuk mengelola data seperti melakukan CRUD (Create, Read, Update, Delete) data dan mampu menyajikan data sebagai bentuk informasi juga dengan harap saya bisa membaca data saya mampu mengambil keputusan dengan baik dan efisien.

ketiga, saya berharap saya dapat memperlajari bagaimana cara membuat database yang rapi dan efisien. tidak hanya sekedar menyimpan data, namun mampun membuat database yang terstruktur terorganisir, tidak boros ruang dan mudah digunakan kembali.

selain itu, saya juga ingin mengetahui bagaimana database dapat diimplementasikan dalam aplikasi nyata. Dengan begitu, saya bisa melihat secara langsung bagaimana peran database dalam menjalankan sebuah sistem.

Dan pada akhirntya, saya ingin mengembangkan fondasi saya terhadap pengetahuan tentang dunia digital terutama database, sehingga saya lebih siap untuk mempelajari mata kuliah selanjutnya maupun tantangan dunia kerja ataupun pada saat mengerjakan suatu project nanti.
